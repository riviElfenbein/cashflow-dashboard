# 🚀 איך להפעיל את הדשבורד לעצמך

מדריך אישי — צעד אחרי צעד.

---

## אפשרות א׳ — הכי מהירה: index.local.html (עם נתונים מוטמעים)

הקובץ `index.local.html` מכיל את כל הנתונים בפנים. לא צריך לעשות כלום.

```
פתחי את הקובץ הזה בכרום:
~/Dev/PersonalDev/cashflow-dashboard/index.local.html
```

זהו. הדשבורד נטען עם כל הנתונים שלך.

> ⚠️ הקובץ הזה **לא מועלה ל-GitHub** (מוגן על ידי `.gitignore`) — הנתונים שלך נשארים פרטיים.

---

## אפשרות ב׳ — העלאת CSV חדש (לאחר יצוא מהבנק / Numbers)

### שלב 1 — יצוא CSV מ-Numbers

1. פתחי את `cashflow 2.numbers` בתוכנת Numbers
2. `File → Export To → CSV`
3. שמרי את הקובץ (למשל בשולחן העבודה)

### שלב 2 — טעינה לדשבורד

1. פתחי את `index.local.html` בכרום (או `index.html` אם אין נתונים מוטמעים)
2. גררי את קובץ ה-CSV לסרגל הצד הימני
3. הדשבורד יטען את הנתונים אוטומטית

### שלב 3 (אופציונלי) — שמירה לסשן הבא

הנתונים נשמרים אוטומטית ב-localStorage של הדפדפן ויופיעו גם בפתיחה הבאה.
אם תרצי לשמור snapshot לקובץ — לחצי **"💾 שמור כ-JSON"** בסרגל הצד.

---

## אפשרות ג׳ — שרת לוקאלי (לצורך פיתוח / בדיקות)

אם הדפדפן חוסם קבצים מקומיים (file:// access errors):

```bash
# פתחי Terminal
cd ~/Dev/PersonalDev/cashflow-dashboard

# הפעילי שרת פשוט
python3 -m http.server 8765

# פתחי בדפדפן:
# http://localhost:8765/index.html          ← גרסה ריקה (ציבורית)
# http://localhost:8765/index.local.html   ← גרסה עם נתונים
```

לעצירת השרת: `Ctrl + C` בטרמינל.

---

## עדכון הנתונים המוטמעים ב-index.local.html

כשיש נתונים חדשים שתרצי לשים בתוך הקובץ עצמו (ולא להעלות בכל פעם מחדש):

```bash
cd ~/Dev/PersonalDev/cashflow-dashboard

# 1. ודאי שיש לך את הספרייה
pip install numbers-parser

# 2. הרצי את הסקריפט (קוראת את cashflow 2.numbers מהשולחן)
python3 extract.py
# → יוצר cashflow_cleaned.json

# 3. עדכני את index.local.html — החליפי את השורה:
#    const RAW = [...]
# בתוכן של cashflow_cleaned.json
```

לאחר מכן, `index.local.html` יכיל את הנתונים החדשים מוטמעים.

## טיפים

- **נתונים נשמרים בדפדפן** — אחרי טעינת CSV לא צריך לטעון שוב. הנתונים ישרדו רענון.
- **למחוק את הנתונים השמורים** — לחצי "מחק נתונים שמורים" בסרגל הצד.
- **עריכת קטגוריה** — לחצי ✏️ בטבלת העסקאות. תוכלי לשנות לכל העסקאות של אותו עסק בבת אחת.
- **מיון** — לחצי על כותרת כל עמודה בטבלה.
- **חיפוש** — תיבת החיפוש מעל הטבלה מסננת לפי שם עסק או קטגוריה.
